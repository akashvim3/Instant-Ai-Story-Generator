Getting Started

Welcome to the AI StoryStream API! Our RESTful API allows you to integrate powerful AI story generation into your applications.

Base URL

https://api.aistorystream.com/v1

Quick Start

Here's a simple example to generate your first story:

curl -X POST https://api.aistorystream.com/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "event_type": "sports",
    "description": "Final goal in championship match",
    "style": "dramatic",
    "length": "medium"
  }'

Authentication

All API requests require authentication using an API key. Include your key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Get your API key from the dashboard after signing up.

Generate Story

Generate AI-powered stories for live streaming events.

POST /v1/generate

Request Parameters

Parameter Type Description
event_typeRequired string Type of event (sports, concert, gaming, news, etc.)
descriptionRequired string Event description (max 500 characters)
styleOptional string Writing style (dramatic, humorous, professional, casual)
lengthOptional string Story length (short, medium, long)
languageOptional string Language code (en, es, fr, etc.)
include_emojisOptional boolean Include emojis in output
include_hashtagsOptional boolean Include hashtags in output

Example Request

const response = await fetch('https://api.aistorystream.com/v1/generate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    event_type: 'sports',
    description: 'Game-winning touchdown in overtime',
    style: 'dramatic',
    length: 'medium',
    language: 'en',
    include_emojis: true,
    include_hashtags: true
  })
});

const data = await response.json();
console.log(data.story);

Translate Story

Translate existing stories to different languages.

POST /v1/translate

Request Parameters

Parameter Type Description
story_idRequired string ID of the story to translate
target_languageRequired string Target language code (es, fr, de, etc.)

Example Request

curl -X POST https://api.aistorystream.com/v1/translate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "story_id": "story_abc123",
    "target_language": "es"
  }'

Webhooks

Set up webhooks to receive real-time notifications when stories are generated.

Configure Webhook

POST /v1/webhooks 
{
  "url": "https://your-domain.com/webhook",
  "events": ["story.generated", "story.translated"],
  "secret": "your_webhook_secret"
}

Error Handling

The API uses standard HTTP status codes:

Status Code Meaning
200 Success
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid API key
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error

Error Response Example

{
  "success": false,
  "error": {
    "code": "invalid_parameter",
    "message": "The 'event_type' parameter is required",
    "param": "event_type"
  }
}

SDKs & Libraries

We offer official SDKs for popular programming languages:

JavaScript/Node.js

npm install @aistorystream/sdk

const AIStoryStream = require('@aistorystream/sdk');
const client = new AIStoryStream('YOUR_API_KEY');

const story = await client.generate({
  event_type: 'sports',
  description: 'Championship goal',
  style: 'dramatic'
});

Python

pip install aistorystream

from aistorystream import Client

client = Client(api_key='YOUR_API_KEY')
story = client.generate(
    event_type='sports',
    description='Championship goal',
    style='dramatic'
)

PHP

composer require aistorystream/php-sdk

use AIStoryStreamClient;

$client = new Client('YOUR_API_KEY');
$story = $client->generate([
    'event_type' => 'sports',
    'description' => 'Championship goal',
    'style' => 'dramatic'
]);
